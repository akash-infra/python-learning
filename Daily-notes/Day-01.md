- There are two types of Modules in python;
  
  **1.Built-In:** They are ready to import which comes along with python interpretor and there is no need to install such modules explicitly.
  
  **2.External Modules:** Writeen by someone else to use opensource.They are writeen as third party file so can be installed using package manager like "pip" or "conda".
